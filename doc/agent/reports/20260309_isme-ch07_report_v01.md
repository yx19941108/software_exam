# 第7章 网络基础知识

## 教材定位
- PDF 目录起始页：122
- 目录关键词：网络体系结构、网络编码与协议、局域网、因特网、网络管理、网络安全

## 你要掌握的核心知识
- 掌握计算机网络的分类、体系结构、分层思想和常见协议。
- 理解传输介质、编码、交换、路由、TCP/IP、局域网与广域网基本原理。
- 掌握 DNS、HTTP、邮件、QoS、网络管理软件等常见网络服务与管理要点。
- 理解网络安全基础、攻击类型和防护思路。

## 常考角度
- 协议分层、地址与编码、TCP/IP、交换路由、网络服务和网络安全是高频客观题。
- 网络题常与系统部署、资源管理、故障定位和安全管理联动出现。
- 下午题里经常借网络场景考查你对系统瓶颈和运维问题的判断。

## 互联网重点标注：中高
- 互联网备考普遍把网络知识归入‘必须稳住的基础分’，尤其是协议、网络服务、网络安全。
- 结合当前系统运维和云化趋势，建议把网络知识放进‘业务可达性、性能、可靠性、安全’四个维度复习。

## 复习建议
- 先吃透协议分层、地址、典型服务，再补网络管理和网络安全场景题。
- 把网络故障定位路径画出来：终端 -> 链路 -> 交换/路由 -> 服务。
