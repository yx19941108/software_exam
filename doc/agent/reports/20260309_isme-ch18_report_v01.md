# 第18章 资源管理

## 教材定位
- PDF 目录起始页：420
- 目录关键词：配置管理、硬件管理、软件管理、网络资源管理、数据管理、机房与设备管理

## 你要掌握的核心知识
- 掌握资源管理的对象、范围和目标，理解配置管理的基础作用。
- 掌握硬件、软件、网络、数据、机房、设备等资源的分类与管理要求。
- 理解资源台账、配置项、变更、资产生命周期和合规管理。
- 掌握资源管理与成本、可用性、安全性和服务质量之间的关系。

## 常考角度
- 配置项、资产台账、资源生命周期、机房环境和设备管理常考。
- 下午题喜欢考资源失控导致的故障、成本、审计和安全问题。
- 资源管理经常和变更管理、容量管理、安全管理一起出现。

## 互联网重点标注：高
- 当前网上备考把资源管理视为运维管理主线之一，优先级很高。
- 结合新版教材和服务管理趋势，建议强化配置管理数据库、资产生命周期、资源可视化与合规审计思维。

## 复习建议
- 复习时把资源对象按硬件、软件、网络、数据、场地五类拆开记。
- 重点理解资源管理为什么是其他管理活动的基础。
