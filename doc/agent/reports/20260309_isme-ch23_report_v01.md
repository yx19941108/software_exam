# 第23章 新系统运行及系统转换

## 教材定位
- PDF 目录起始页：512
- 目录关键词：运行计划、运行体制、系统转换测试、系统转换、开发环境管理

## 你要掌握的核心知识
- 掌握新系统运行前的计划制定、组织准备和运行体制建设。
- 理解系统转换测试、试运行、并行切换、直接切换等转换方式及风险。
- 掌握系统转换中的数据、流程、人员和环境准备要点。
- 理解开发环境配置、版本管理和发布控制的重要性。

## 常考角度
- 切换方式、上线准备、试运行、回退预案和发布控制是高频点。
- 下午题喜欢考如何保证业务连续性和降低上线风险。
- 本章与实施、维护、用户支持和评价章节衔接紧密。

## 互联网重点标注：中高
- 近年网上备考把系统切换和上线管理视为容易出综合场景题的内容。
- 结合当前机考和服务连续性导向，建议重点掌握切换策略选择、风险控制、验证与回退。

## 复习建议
- 把几种切换方式做优缺点对比表，能快速应对选择题和简答题。
- 答题时一定补上回退、通知、培训、验证和数据校验。
