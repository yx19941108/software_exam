# 第5章 数据结构与算法

## 教材定位
- PDF 目录起始页：83
- 目录关键词：线性结构、树和图、查找、排序、算法复杂度

## 你要掌握的核心知识
- 掌握线性表、栈、队列、串、数组、树、图等基本数据结构及应用场景。
- 理解查找、排序、遍历、递归等常见算法思想，以及时间复杂度和空间复杂度。
- 能够从问题特征出发选择合适的数据组织方式和算法策略。
- 掌握数据结构与系统分析、数据库设计、程序设计之间的关系。

## 常考角度
- 上午题稳定考点包括复杂度、树和图、排序查找、栈队列、递归。
- 下午题不直接深挖算法，但分析设计时常隐含数据组织与处理流程选择。
- 需要会用‘结构特征 + 操作需求’去判断适用的数据结构。

## 互联网重点标注：中高
- 网上备考普遍把本章视为上午题性价比很高的基础得分点。
- 新版教材虽然更偏管理，但分析设计和数据库部分仍要求考生具备基本的数据抽象能力。

## 复习建议
- 至少把复杂度、树/图、排序查找这些经典考点刷熟，保证客观题得分。
- 复习时不要只背定义，要能说出某种结构为什么适合某类业务场景。
