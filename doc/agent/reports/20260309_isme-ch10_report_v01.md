# 第10章 信息系统开发的基础知识

## 教材定位
- PDF 目录起始页：206
- 目录关键词：信息系统概念、生命周期、开发阶段、开发方式、开发方法、开发工具

## 你要掌握的核心知识
- 掌握信息系统的概念、结构、分类和业务支撑作用。
- 理解信息系统生命周期、开发阶段、开发方式和典型开发方法。
- 掌握结构化、原型化、面向对象等方法的特点、适用场景和优缺点。
- 理解开发工具在需求、设计、编码、测试、配置和管理中的作用。

## 常考角度
- 生命周期、开发模型、方法对比、开发工具定位是高频考点。
- 下午题经常要求根据组织环境选择合适的开发方式或开发模型。
- 要会把‘项目特点 -> 开发策略 -> 风险与控制点’说清楚。

## 互联网重点标注：高
- 近年网上备考普遍把系统开发全流程列为本资格考试的主线之一。
- 新版教材强调信息系统与数字化、系统架构和服务化思维，复习时要把传统开发模型升级为面向业务价值和持续交付的理解。

## 复习建议
- 把常见开发模型做成对比表，重点看适用场景、优缺点和风险控制。
- 复习时同步思考需求变更、沟通成本、质量控制这些管理因素。
