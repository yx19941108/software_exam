# 第2章 操作系统知识

## 教材定位
- PDF 目录起始页：24
- 目录关键词：进程管理、存储管理、设备管理、文件系统、作业管理

## 你要掌握的核心知识
- 掌握操作系统的定义、功能和特征，理解其在资源管理与用户接口中的作用。
- 理解进程与线程、同步互斥、调度、死锁、存储管理、分页分段等核心概念。
- 掌握文件系统结构、设备管理和作业管理的基本机制及其管理目标。
- 能够从吞吐量、周转时间、利用率等指标角度分析操作系统行为。

## 常考角度
- 进程与线程、PV 操作、死锁条件、分页分段、虚拟存储是高频基础题。
- 文件系统与设备管理常结合管理目标出题，如共享、保护、调度、公平性。
- 下午分析题虽然不深考 OS，但会借用进程、存储、调度概念解释系统现象。

## 互联网重点标注：中
- 近年备考资料强调操作系统知识要为系统性能分析、资源配置和故障处理服务，而不是只背定义。
- 新版考试更关注虚拟化、资源池化背景下的操作系统认知，因此要会把传统概念迁移到云环境。
