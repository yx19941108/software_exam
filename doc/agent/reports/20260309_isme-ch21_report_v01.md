# 第21章 性能及能力管理

## 教材定位
- PDF 目录起始页：483
- 目录关键词：性能管理、能力管理、监控指标、基线、容量规划、评估报告

## 你要掌握的核心知识
- 掌握性能管理和能力管理的定义、目标及其区别。
- 理解性能监控、性能指标、基线、趋势分析和性能调优的基本方法。
- 掌握能力管理、容量规划、需求预测、能力评估和验证流程。
- 理解性能与容量管理如何支撑服务水平和成本优化。

## 常考角度
- 性能指标、基线、瓶颈分析、容量规划和验证流程是高频考点。
- 下午题常考如何建立性能监控体系，以及如何做扩容和容量预测。
- 这章经常与资源管理、服务等级、成本管理合并考查。

## 互联网重点标注：高
