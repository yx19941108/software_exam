# 第4章 系统配置和方法

## 教材定位
- PDF 目录起始页：67
- 目录关键词：系统结构、系统性能、系统配置、开放性、可靠性

## 你要掌握的核心知识
- 掌握系统结构、配置方法、开放性、可扩展性、可靠性等系统级概念。
- 理解系统性能指标、瓶颈识别和配置优化的基本思路。
- 掌握系统配置与部署时需要考虑的硬件、软件、网络和安全因素。
- 能从成本、性能、可靠性和可维护性角度评价配置方案。

## 常考角度
- 上午题常考系统开放性、可靠性设计、性能指标、配置原则等概念辨析。
- 这章与后续系统设计、资源管理、性能管理、运维管理联系很紧。
- 题目喜欢把选型原则和场景约束绑定在一起考。

## 互联网重点标注：中
- 近年网上重点把本章映射到架构设计与部署选型，强调从业务目标反推资源配置。
- 新版教材更重视云服务、分层架构和系统治理，因此复习时要主动把传统配置知识升级为架构思维。

## 复习建议
- 把常见质量属性整理成表：性能、可靠性、可扩展性、可维护性、开放性、安全性。
- 看到配置题时先想目标和约束，再判断方案，不要只记名词。
