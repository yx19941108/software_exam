# 第1章 计算机硬件基础

## 教材定位
- PDF 目录起始页：2
- 目录关键词：计算机基本组成、CPU、存储器、I/O设备、系统结构、并行与多媒体

## 你要掌握的核心知识
- 掌握冯·诺依曼结构、CPU 构成、指令执行过程，以及主频、字长、总线等性能指标。
- 理解寄存器、Cache、主存、辅存的层次关系，会比较容量、速度、成本和适用场景。
- 掌握 I/O 控制、中断、DMA、通道、总线等基本机制，能看懂设备访问路径。
- 理解 CISC/RISC、流水线、并行处理、多媒体与人工智能硬件基础概念。

## 常考角度
- 上午题常把性能指标、存储层次、I/O 控制方式和可靠性放在同一道题里考。
- 容易混淆的对比点包括 CISC 与 RISC、并行与分布式、主存与辅存、DMA 与中断。
- 硬件知识常与后续的资源管理、性能管理、故障定位场景联动出现。

## 互联网重点标注：中
- 网上备考普遍认为纯硬件细节占比不如开发管理和运维管理高，但它仍然是上午题稳定基础分。
- 结合新版教材趋势，建议把硬件知识放到系统部署、容量规划、性能分析和故障定位场景中理解。

## 复习建议
- 按“组成 -> 指标 -> 场景 -> 常见陷阱”做表格，背概念时同时记适用场景。
- 把 Cache/主存/辅存、DMA/中断、并行/分布式放在一页上对比记忆。
