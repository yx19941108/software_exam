# 第24章 信息系统评价

## 教材定位
- PDF 目录起始页：524
- 目录关键词：技术性能评价、管理效益评价、成本与经济效益评价、运行目标评价、改进建议

## 你要掌握的核心知识
- 掌握信息系统评价的概念、目的和评价维度。
- 理解技术性能、管理效益、经济效益和综合价值等评价内容。
- 掌握评价指标设计、评价过程、评价标准和改进建议形成方法。
- 理解评价活动与决策支持、持续改进和投资回报分析之间的关系。

## 常考角度
- 评价指标、评价维度、经济性与效益分析、改进建议是常考点。
- 下午题常从建设后评估、运行效果、成本收益和改进方向切入。
- 本章容易和信息化建设、系统管理、用户支持一起考。

## 互联网重点标注：高
- 根据网上对新版教材的解读，评价不再只是结项动作，而是数字化价值闭环的重要环节。
- 复习时建议把本章理解为‘目标 -> 指标 -> 采集 -> 评价 -> 改进’全过程。

## 复习建议
- 把技术、管理、经济、综合四类评价维度分开整理，防止答题泛化。
- 看到评价题时要补上可量化指标和改进建议。
