# 第6章 多媒体基础知识

## 教材定位
- PDF 目录起始页：107
- 目录关键词：媒体类型、图像、音频、视频、压缩编码、应用系统

## 你要掌握的核心知识
- 掌握媒体、超媒体、多媒体系统的基本概念及其组成。
- 理解图像、音频、视频的数字化表示方式，以及分辨率、采样率、色彩深度等指标。
- 掌握常见压缩编码的基本思想和应用场景。
- 了解多媒体应用系统的构成与常见处理流程。

## 常考角度
- 上午题常考采样率、分辨率、压缩、媒体类型分类等基础概念。
- 通常不会深考，属于低成本补分章节。
- 题目容易把图像、音频、视频的指标混在一起考，需要注意单位和场景。

## 互联网重点标注：低
- 综合网上备考资料，本章不是当前资格考试的主战场，但仍可能作为上午题零散考点出现。
- 建议以能做题、会辨析概念为目标，不必投入过多时间。

## 复习建议
- 记住几个最常考指标：分辨率、采样率、色彩深度、压缩方式。
- 用近年真题或模拟题快速过一遍即可。
