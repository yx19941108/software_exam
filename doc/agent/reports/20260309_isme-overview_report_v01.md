# 信息系统管理工程师教材章节总览

## 说明
- 本套资料基于本地扫描版 PDF `01、信息系统管理工程师教程.pdf` 的目录页与章节结构整理。
- 由于 PDF 为扫描版，直接抽取正文文本受限，因此本次总结以“教材目录结构 + 章节主题归纳 + 互联网最新备考重点”结合输出。
- 互联网重点标注参考了 2026-03-09 可检索到的官方考试安排、机考通知，以及新版教材/备考方向解读。

## 当前备考要点
- 官方网站显示，`信息系统管理工程师` 仍在软考中级资格列表内，可通过中国计算机技术职业资格网关注报名与安排。
- 深圳市人力资源和社会保障局发布的 2026 年度下半年计算机软件资格考试通知显示，下半年考试时间为 `2026-10-24` 至 `2026-10-27`，其中包含 `信息系统管理工程师`。
- 上海市职业能力考试院发布的 2025 年软考通知明确提到该考试采取 `计算机化考试` 形式，说明机考已是备考时必须适应的基本环境。

## 新版/近年互联网重点
- 高优先级主线 1：系统开发全过程。重点对应第 `10` 至 `14` 章，尤其是需求分析、系统设计、测试与实施。
- 高优先级主线 2：系统管理与 IT 服务。重点对应第 `16` 至 `23` 章和第 `25` 章，尤其是资源、故障、安全、性能、维护、用户支持。
- 高优先级主线 3：信息化、标准化、评价。重点对应第 `15` 章和第 `24` 章，近年与数字化转型、治理和价值评估联系更紧。
- 基础得分区：第 `1` 至 `9` 章仍然重要，但更适合作为上午题基础和下午题背景知识，不宜孤立背诵。

## 学习顺序建议
1. 先打主线：第 `10` 至 `15` 章。
2. 再攻运维治理：第 `16` 至 `21` 章、`25` 章。
3. 然后补交付与评价：第 `22` 至 `24` 章。
4. 最后回补基础：第 `1` 至 `9` 章。
