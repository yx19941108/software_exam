# 第9章 安全性知识

## 教材定位
- PDF 目录起始页：189
- 目录关键词：安全性简介、访问控制、完整性、可用性、计算机病毒、安全分析与管理

## 你要掌握的核心知识
- 掌握保密性、完整性、可用性、可控性、不可否认性等信息安全基本属性。
- 理解访问控制、身份认证、权限管理、审计、完整性保护和可用性保障思路。
- 掌握计算机病毒、恶意代码和安全风险分析的基本概念。
- 理解从技术、管理、制度三个层面构建安全体系的基本框架。

## 常考角度
- 上午题常考安全属性、访问控制模型、风险、审计、病毒和攻击防护。
- 这章是后续安全管理章的基础，下午题经常要你把安全要求落到制度和流程上。
- 安全题越来越强调管理和合规，不只是技术名词。

## 互联网重点标注：高
- 网上近年备考一致把安全列为重点章节，尤其是在机考背景下，场景化安全管理题更常见。
- 结合新版教材和当前考试趋势，建议把本章与第20章安全管理一起复习，形成‘基础概念 + 管理措施’闭环。

## 复习建议
- 把安全属性、控制方式、风险处理、审计与应急响应串成一张图。
- 遇到安全题时优先从技术、管理、制度三个层面分解。
