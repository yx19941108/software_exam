# 第22章 系统维护

## 教材定位
- PDF 目录起始页：502
- 目录关键词：维护任务、制度计划、维护过程、软件维护、硬件维护

## 你要掌握的核心知识
- 掌握系统维护的任务、内容、方式和维护计划。
- 理解纠错性、适应性、完善性、预防性等维护类型。
- 掌握维护工作的流程、组织分工、记录与评估要求。
- 理解软件维护与硬件维护的差异及其管理重点。

## 常考角度
- 维护类型、维护计划、维护流程和维护文档是常考点。
- 下午题常要求说明如何降低维护风险、如何安排停机与回退。
- 维护题通常和变更管理、版本管理、用户支持一起出现。

## 互联网重点标注：中高
- 网上备考一般把系统维护列为运维章节的中高优先级内容。
- 结合新版教材更强调持续运行和服务质量，建议把维护活动放到配置、变更、版本和支持体系中整体理解。

## 复习建议
- 把维护类型和对应目标、典型场景整理成表格。
- 答题时要体现维护前评估、实施、验证、记录、回退五个动作。
