# 第17章 系统管理综述

## 教材定位
- PDF 目录起始页：392
- 目录关键词：系统管理分类、IT组织与岗位、日志管理、用户管理、运行工具、ITIL/COBIT/MOF

## 你要掌握的核心知识
- 掌握系统管理的分类、范围和总体目标，理解 IT 服务管理的全局框架。
- 理解 IT 人员组织、职责划分、外包协同、系统日志、用户管理和运行工具的管理要求。
- 掌握 ITIL、COBIT、MOF 等框架的基本定位和差异。
- 理解分布式系统环境下系统管理的新特点。

## 常考角度
- ITIL/COBIT/MOF、组织职责、日志管理、运行工具和成本管理是高频点。
- 这章常作为下午题的理论支撑，要求你把管理框架落到场景中。
- 答题时要体现‘流程化、制度化、可追踪、可度量’。

## 互联网重点标注：高
- 网上备考一致认为第三篇系统管理是本资格考试最有区分度的板块之一。
- 结合新版教材强调信息技术服务与治理，本章要重点掌握管理框架、角色分工和可度量管理思维。

## 复习建议
- 把 ITIL、COBIT、MOF 的定位和适用场景做成简表，避免混淆。
