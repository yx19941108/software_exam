# 第8章 数据库技术

## 教材定位
- PDF 目录起始页：160
- 目录关键词：数据库基础、关系数据库、SQL、数据库管理系统、控制功能

## 你要掌握的核心知识
- 掌握数据库系统、数据库管理系统和数据模型的基本概念。
- 理解关系模型、主键外键、关系运算、完整性约束和规范化基础。
- 掌握 SQL 基本语句和数据库控制功能，如并发控制、恢复、备份、安全。
- 理解数据库在系统分析、设计、实施和评价中的核心地位。

## 常考角度
- 关系模型、SQL、主键外键、事务并发、恢复与安全是稳定高频点。
- 数据库题既会出现在上午题，也会渗透到下午的分析设计、实施和运维场景。
- 要会把数据库概念转换成业务语义，而不是只会写语句。

## 互联网重点标注：中高
- 网上备考通常把数据库和系统分析设计放在同一优先级，因为它直接影响下午题表达。
- 结合新版教材趋势，建议强化‘数据资源管理、数据安全、数据质量、数据价值’的管理视角。
