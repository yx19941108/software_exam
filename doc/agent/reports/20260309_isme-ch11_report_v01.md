# 第11章 信息系统开发的管理知识

## 教材定位
- PDF 目录起始页：228
- 目录关键词：项目概念、项目过程、常用管理工具、计划管理、跟踪控制

## 你要掌握的核心知识
- 掌握信息系统项目的概念、目标、过程和主要管理对象。
- 理解进度、范围、成本、质量、沟通、风险、配置等项目管理核心内容。
- 掌握项目计划、执行、监控和收尾的基本思路。
- 理解开发管理工具在计划、缺陷、配置和跟踪中的作用。

## 常考角度
- 项目管理过程、进度计划、风险控制、变更管理和质量保证是下午题常考点。
- 旧教材中的具体工具名称较老，但工具背后的管理目的仍然高频。
- 题目常要求你从项目经理视角说明为什么这样管、如何落地。

## 互联网重点标注：高
- 网上备考普遍把项目管理和系统分析设计一起视为下午题核心拿分区。
- 结合新版教材趋势，应弱化过时工具名词，强化计划、协同、缺陷、配置、风险和验收这些管理动作。

## 复习建议
- 重点复习管理对象、管理过程、常见文档和控制点，不必纠结老工具界面细节。
- 用‘问题 -> 管理措施 -> 预期效果’的结构练习简答题。
