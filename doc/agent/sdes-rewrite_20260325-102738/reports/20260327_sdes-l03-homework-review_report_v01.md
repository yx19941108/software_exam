- 失分原因：

  - 你的抱怨不能替代答案，按考试口径记 `0`
- 事实校对：
  - 课案并非只写了“左根右”
  - `L03` 已明确给出“沿左孩子下探 -> 路上结点压栈 -> 到头后弹栈访问根 -> 再转向右子树”，并指出栈保存的是“稍后还要回来处理”的祖先路径
- 标准答案摘要：
  - 从根开始，一直往左走
  - 路上经过的每个结点先压栈，因为这些结点当前还不能访问
  - 走到最左端后，弹出栈顶并访问该结点
  - 然后转去处理这个结点的右子树
  - 对右子树重复同样过程，直到当前结点为空且栈也为空
  - 栈中保存的是“已经遇到但左子树还没完全处理完、之后还要回来访问和转向右子树的祖先结点”

### 练习 7

- 判定：未达标
- 建议分：`0 / 5`
- 失分原因：
  - 没有作答
- 标准答案摘要：
  - “自底向上”说明问题是从小规模结果逐步推出大规模结果
  - “保存中间结果”说明相同子问题不会被反复重算
  - 这正是动态规划的核心：把重复子问题的答案存下来复用
  - 暴力枚举或普通递归搜索通常会把大量相同子问题一遍遍重算，不强调状态复用
  - 所以这两个信号放在一起，往往提示是动态规划而不是暴力枚举

### 练习 8

- 判定：部分正确
- 建议分：`4 / 5`
- 已答对：
  - 你抓住了“复杂度看增长级别，不看精确次数”
  - 你知道高阶项主导，所以总复杂度仍是 `O(n^2)`
- 失分点：
  - “指数级别比较低”这个说法错误，这里不是在比较“指数级”
  - 更准确的说法应是：`O(n) + O(n^2) + O(n)` 中，二次项增长最快，因此总复杂度写成 `O(n^2)`
