# 软件设计师课程重写版教学标准决议

## Decision

自本次重写起，软件设计师课程的 `L02/L03` 以及后续课案统一采用以下标准：

1. 课案必须可独立承担主学习材料，不默认假设用户已经读过教材。
2. 课案必须优先按真题方向组织内容，而不是按教材章节顺序平铺。
3. 课后习题必须尽可能贴近真题，并标注：
   - 单题分值
   - 样本频次
   - 频次系数
   - 权重
4. 习题批改采用严格考试口径：
   - 术语不准，扣分
   - 因果链不完整，扣分
   - 审题没扣准，不算全对
   - 只写结论不写依据，最多算半对

## Weight Formula

### 字段定义

- `样本频次`: 在当前已验证样本窗口中，出现该考法的试卷数
- `频次系数`:
  - `4`: 当前样本中出现 `6+` 次，或可证实“几乎年年出现”
  - `3`: 当前样本中出现 `3-5` 次
  - `2`: 当前样本中出现 `1-2` 次
  - `1`: 当前样本未直接命中，但属于后续高频考法的前置基础
- `权重 = 单题分值 × 频次系数`

### 解释

- 权重不是“预测必考分”，而是帮助用户判断优先级。
- 若某题权重相同，则优先做更贴近真题原型的那道。
