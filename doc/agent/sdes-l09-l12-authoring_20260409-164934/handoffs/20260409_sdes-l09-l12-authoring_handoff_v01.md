## 各课重点提醒

### L09 设计模式 Java 路线

- 不要只列模式定义。
- 要讲清：
  - 为什么会出现某模式
  - 什么时候不用它
  - 题里如何从“类关系/职责变化/代码结构”识别它
- Java 路线必须明确。

### L10 算法与代码题 II

- 不能只当作 L03 的复读。
- 要把下午算法题常见问法讲成模板：
  - 读代码
  - 补空
  - 推状态转移
  - 看图算法
  - 估复杂度

### L11 软件工程与测试

- 这是上午高频综合模块，不要写成薄薄的名词索引。
- 要覆盖：
  - 生命周期模型
  - 需求/设计/维护
  - 测试分层与方法
  - 白盒/黑盒
  - 质量、配置管理、项目管理常见考法

### L12 操作系统 / 网络 / 安全

- 这是压缩型大模块，但不能压缩成空壳。
- 必须把高频点讲透：
  - 进程线程
  - 调度与死锁
  - 存储管理
  - 网络分层与常见协议
  - 地址/路由/可靠传输
  - 加密、摘要、签名、常见攻击
