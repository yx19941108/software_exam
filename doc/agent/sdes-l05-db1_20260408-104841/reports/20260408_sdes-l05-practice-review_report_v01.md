## 课后作业批改

### 作业 1：一句话定义 1NF / 2NF / 3NF

- 判定：部分正确
- 说明：
  - `2NF` 和 `3NF` 基本方向对
  - `1NF` 错了
- 为什么错：
  - `1NF` 不是“所有表都必须有一个主键”
  - `1NF` 的关键是属性值原子化、无重复组、一个单元格里不再塞可分项
- 建议改成：
  - `1NF：每个字段都应是不可再分的单值，不出现重复组`

### 作业 2：四张表键关系图

- 判定：方向对，但未达考试化标准
- 说明：
  - 你已经表达出了：
    - `订单表` 通过 `客户编号` 关联 `客户表`
    - `订单明细表` 通过 `订单号` 关联 `订单表`
    - `订单明细表` 通过 `商品编号` 关联 `商品表`
  - 但图还存在几个问题：
    1. 没明确标出哪些是主键、哪些是外键
    2. 没明确说明 `订单明细` 的复合主键通常是 `(订单号, 商品编号)`
    3. 图形语义偏自由发挥，不是最稳的考试化表达
    4. 本课建议图示优先用 `Mermaid`，你这次是图片表达，能看懂，但不够标准化
