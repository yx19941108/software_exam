# 软件设计师课程总控文档

## 目标

- 用户：游工
- 目标考试：2026 年 5 月上半年软件设计师考试
- 核心目标：稳过线，并尽可能提高总分
- 当前基础：零散基础
- 学习投入：每周约 10-14 小时

## 已确认默认项

- 教学路线：双线螺旋
- 单节时长：60-90 分钟
- 交付模式：分节推进
- 图示语言：Mermaid
- 设计模式语言路线：Java
- 训练强度：每课有作业 + 阶段测

## 阶段目标

### 阶段 A：L01-L04

- 建立考试全景
- 完成上午快得分模块入门
- 建立 DFD 基本模板

### 阶段 B：L05-L08

- 建立数据库与 UML 的固定题框架
- 下午三大稳定分项成型

### 阶段 C：L09-L12

- 固定下午剩余高频题型
- 打通上午核心基础模块

### 阶段 D：L13-L16

- 上午查漏补缺
- 下午综合串题
- 全真模拟冲刺

## 每节课固定结构

1. 学习目标
2. 前置知识
3. 核心概念
4. 真题例题
5. Mermaid 图
6. 随堂练习
7. 课后作业
8. 易错点
9. 复盘清单

## 跨会话执行约定

1. 新会话继续软件设计师课程前，先读取本文件。
2. 再读取最新进度台账与最新交接单。
3. 先复述当前状态，再进入新教学内容。
4. 每节课结束后，必须更新进度台账和交接单。
