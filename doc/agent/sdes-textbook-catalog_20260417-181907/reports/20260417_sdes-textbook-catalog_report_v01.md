# 软件设计师教程（第5版）在线来源判断与目录固化

## 1. 结论

目前已确认：

1. 书的正式信息明确无误：
   - 书名：`软件设计师教程（第5版）`
   - ISBN：`9787302491224`
   - 出版日期：`2018-02-01`
   - 出版社：`清华大学出版社`
2. 官方页面明确提供了电子书入口，但在当前浏览工具下，我无法稳定进入可逐页阅读的全文界面。
3. 我已经找到可稳定读取的完整目录来源，因此后续不必再为“这本书到底有哪些章”反复解析本地 PDF。

## 2. 关于“是否找到可读在线教材”的判断

### 2.1 已确认的可用来源

- 官方书目页：
  - `https://www.tup.tsinghua.edu.cn/bookscenter/book_07552001.html`
  - 可确认书名、ISBN、出版信息、电子书入口、内容简介，以及至少前半部分目录。
- 目录转载页：
  - `https://www.cnblogs.com/jiftle/p/10686928.html`
  - 可直接读取完整目录，覆盖全书 `12` 章。

### 2.2 暂未确认的部分

- 官方页面的电子书入口指向 `wqbook.com`，但在当前环境中无法稳定打开到可连续阅读的正文页。
- 因此，我不能把它当成“已经找到我能稳定通读的在线全文教材”。

### 2.3 当前最稳方案

- 把第 5 版教材目录固化成 Markdown。
- 后续按目录定位章节，再结合本地题库与其它可读资料推进，不再强依赖当前这份 PDF 的文本抽取能力。

## 3. 目录使用说明

- 本文优先固化“章目录”和“节目录”。
- 若后续需要，我可以继续把某一章扩成“章节知识点导航版”。
- 目录以官方清华社页面为主、转载页为补足。
- 注意：转载页中第 1 章标题曾出现疑似录入错误“计算机网络概论”，但其小节内容与官方页面一致，均指向“计算机系统知识”，因此本文按官方标题记录。
