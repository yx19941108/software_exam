## 本轮已覆盖知识点

### 1. 存储体系与总线

- 系统总线用于连接 CPU、主存、I/O 与外设接口等系统级部件。
- 硬盘属于辅存，不属于主存、缓存或寄存器。
- 在多级存储体系里，DMA/中断控制器面向 CPU 提供的是主存物理地址，不是虚拟地址或硬盘扇区地址。

### 2. DMA 与中断

- DMA 的核心是由 DMA 控制器在主存与外设之间直接搬运数据，CPU 只在开始和结束时参与。
- 中断的本质是 CPU 暂停当前程序，转去处理中断服务，结束后返回原程序继续执行。
- 中断向量提供的是中断服务程序入口地址。
- 中断方式的特点包括：
  - 需要保存和恢复现场
  - 能及时响应突发事件
  - CPU 不必持续轮询外设
- “CPU 主动查询外设状态”属于程序查询方式，不属于中断方式。

### 2.1 三种输入输出控制方式对比

- 程序查询控制方式：
  - CPU 主动反复查询外设状态
  - 实现简单
  - CPU 开销大
  - 适合低速、简单外设
- 中断控制方式：
  - 外设准备好后主动向 CPU 发中断请求
  - CPU 在需要时才介入
  - 需要现场保护与恢复
  - 适合随机事件、突发事件响应
- DMA 控制方式：
  - 数据在主存与外设之间直接传输
  - CPU 只负责启动、结束等控制
  - 更适合大批量数据传输

### 2.2 高频易混点

- 程序查询方式的关键词：
  - `CPU 主动查询`
  - `忙等`
- 中断方式的关键词：
  - `及时响应`
  - `保存/恢复现场`
  - `中断向量`
- DMA 方式的关键词：
  - `主存与外设直接交换数据`
  - `CPU 不逐字节搬运`

### 2.3 本轮补充记忆结论

- 若题目强调“CPU 一直问外设好了没有”，优先想到程序查询方式。
- 若题目强调“突发事件、现场保护、服务程序入口”，优先想到中断方式。
- 若题目强调“大块数据、高效率、主存与外设直连传输”，优先想到 DMA 方式。

### 3. 定点数、补码与浮点数

- 定点数常用补码表示。
- 补码的关键特征：
  - 零的表示唯一
  - 可将减法转化为加法
  - 符号位可与数值位一起参加运算
- 对于定点纯小数，只有补码可以精确表示 `-1`。
- 浮点数加法前必须先对阶：
  - 小阶向大阶对齐
  - 尾数右移

### 4. 流水线

- 流水线吞吐率取决于最长流水段操作时间。
- 计算总执行时间时要区分：
  - 首条指令完整通过流水线的总延迟
  - 后续指令按节拍推进
- 当各段时间不相等时，节拍由最长段决定。

### 5. CRC 与海明码

- CRC 使用生成多项式构造校验码。
- CRC 码的常见格式是：
  - `n` 个数据位之后跟 `k` 个校验位
- 海明码用于发现并纠正 1 位错误。
- 海明码校验位公式：
  - 若数据位为 `m`，校验位为 `r`
  - 则满足 `2^r >= m + r + 1`
- 对 `8` 位信息位，至少需要 `4` 位校验位。
- 对 `32` 位数据构造海明码时，至少需要 `6` 位校验位。

### 6. 可靠性

- 串并联系统可靠性题要先看结构，再写表达式。
- 并联子系统可靠性：
  - `1 - (1 - R)^n`
- 串联系统可靠性：
  - 各部件可靠性相乘
- 真题里常考“先并后串”“先串后并”的混合结构识别。

### 7. RISC / CISC

- RISC 特征：
  - 指令少、简单
  - 指令长度常固定
  - 寻址方式较少
  - 寄存器较多
  - 便于流水线
  - 常配合硬布线控制
- CISC 特征：
  - 指令复杂
  - 寻址方式多
  - 一条指令可能完成更多复杂功能
- 高频误区：
  - 把“指令功能强大、寻址方式复杂多样”误选为 RISC 特征

### 8. 密码学基础与 HTTPS

- DES 属于对称加密算法。
- SHA 属于 Hash / 摘要算法。
- 数字证书用于身份认证，本质是“身份信息与公钥的绑定证明”。
- 数字签名用于保证：
  - 身份真实性
  - 内容完整性
  - 不可否认性
- HTTPS / SSL-TLS 的两阶段：
  - 握手阶段：证书、公钥、密钥交换
  - 传输阶段：会话密钥 + 对称加密
- 高频误区：
  - 误以为 HTTPS 传输阶段一直使用公钥算法加密消息本体

### 9. 网络安全基础

- 应用级网关防火墙能够对应用层通信数据流进行监控和过滤。
- Web 应用防火墙不能有效防护流氓软件。
- 访问控制三大经典模型：
  - DAC：自主访问控制
  - MAC：强制访问控制
  - RBAC：基于角色的访问控制
- 入侵检测与入侵防御区分：
  - IDS 偏检测和告警
  - IPS 偏检测并主动阻断

### 10. 信息安全基本要素

- 已覆盖的基本要素判断：
  - 保密性
  - 完整性
  - 可控性
- “可重用性”不属于信息安全基本要素。
