# 第4章第三轮真题训练

## 使用说明

1. 本轮共 `7` 题，合计 `10` 个空。
2. 这是第 `4` 章补弱训练，重点只打这 `4` 类薄弱点：
   - 进程资源图 / 死锁
   - 路径树图
   - 磁盘旋转 / 磁盘调度

4. 组合题请按 `题号-问题号` 作答，例如：`3-1:A 3-2:B`。
