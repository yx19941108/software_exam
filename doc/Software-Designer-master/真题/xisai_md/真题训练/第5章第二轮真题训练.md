# 第5章第二轮真题训练

## 使用说明

1. 本轮共 `20` 题，其中包含 `1` 道组合题，共 `21` 个空。
2. 本轮继续围绕第 `5` 章《软件工程基础知识》，并对上一轮薄弱点做了加压：`可维护性`、`质量模型子特性`、`过程模型选择`、`阶段边界`、`配置管理`、`UML基础`。
3. 本文件不附答案。
4. 普通题请按 `1:A 2:B 3:C` 的格式作答；组合题请按 `18-1:A 18-2:C` 的格式作答。
