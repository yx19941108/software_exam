# 第5章第三轮真题训练

## 使用说明

1. 本轮共 `10` 题，定位为第 `5` 章基础定义补弱训练。
2. 重点只打这几类还会反复丢分的点：`软件工程基本要素`、`概要设计职责`、`正式技术评审`、`原型模型适用边界`、`易使用性子特性`、`需求分析产物`、`配置管理边界`、`UML用例图参与者`。
3. 本文件不附答案。
4. 请直接按 `1:A 2:B 3:C` 的格式作答。
