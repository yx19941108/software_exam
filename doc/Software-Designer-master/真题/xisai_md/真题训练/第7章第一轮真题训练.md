# 第7章第一轮真题训练

## 使用说明

1. 本轮共 `20` 题，均为第 `7` 章《面向对象技术》相关的上午真题。
2. 覆盖重点包括：面向对象基础概念与分析设计活动、继承与多态、动态绑定、常见设计原则、UML 常见图与关系、典型设计模式。
3. 题目优先选取近 `10` 年本地真题，不硬凑偏题。
4. 本文件不附答案。
5. 请直接按 `1:A 2:B 3:C` 的格式作答。
6. 后续我会严格按本训练文件原题批改，并把做错题与详细讲解题按仓库规则并入 `第7章_面向对象技术.md`。
