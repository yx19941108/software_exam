# 第5章第一轮真题训练

## 使用说明

1. 本轮共 `20` 题，均为第 `5` 章《软件工程基础知识》相关的上午真题。
2. 覆盖重点包括：软件过程模型、需求分析、系统设计、系统测试、运行维护、项目管理、软件质量、UML。
